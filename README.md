# Ninja Jo Artworks — Website

Static marketing site for [ninjajoartworks.com](https://ninjajoartworks.com).

Deployed via Cloudflare Pages (Git-connected).

## Local preview

```bash
npm run preview
```

## Deploy

Push to `main` — Cloudflare Pages builds automatically.

Manual fallback from this folder:

```bash
npm run deploy   # requires CLOUDFLARE_API_TOKEN
```
